# Superstore Sales Performance Dashboard

A sales dashboard built with Power BI using the Superstore dataset from Kaggle.

## Overview

This is my first Power BI project. I used it to practice building a data model, writing DAX measures, and designing an interactive dashboard across multiple pages.

## Dashboard Pages

**Overview** — High-level KPIs including Total Sales, Total Profit, Profit Margin %, and Total Orders. Includes a sales trend line chart, regional breakdown, and category distribution.

**Products** — Sub-category sales comparison, discount vs. profit scatter plot, and profit by category.

**Customers** — Top 10 customers by sales, segment breakdown, and shipping mode analysis.

## DAX Measures

- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Average Order Value

## Tools Used

- Power BI Desktop
- Superstore Dataset (Kaggle)
